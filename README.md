## About me...

I'm Ricardo, a front-end web developer & graphic designer. I've studied Communcation & Multimedia Design at HAN University of Applied Sciences and obtained my Bachelor of Science in February, 2019. I have an intense love for digital design, front-end web development and photography. Knowing that I both love graphic design and technology, I chose to combine those passions and become a front-end web developer. I've been exploring Vue, TypeScript and Golang lately and built some things with these tech stacks.

### Projects I'm currently working on

- [Lighthouse Lab][], a organisation that aims to develop free and open-source software for the web and beyond.
- [Waarzitje.nu][], a modern API-driven location tracker built with Go 1.14, Vue 3 and TypeScript.
- [My personal website][], because there are many [articles to be written][Ricardo Balk's list of articles] and there's always [room for improvement][Ricardo Balk's list of improvements].
- ... and graphic design-related things (i.e. I'm planning to starting my own webshop)
