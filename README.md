# Hi, I'm Ricardo 👋 

I’m a passionate full-stack software engineer who enjoys building efficient, scalable, and user-friendly applications. With experience in both frontend and backend technologies, I thrive on tackling complex challenges and crafting creative, effective solutions.

In my current role, I collaborate closely with data, backend, frontend, and platform specialists to ensure seamless integration across our multi-tenant, containerized infrastructure. Beyond contributing to the codebase, I focus on optimizing interactions between different system components. Within our team, we embrace challenges with confidence &mdash; reminding each other that there’s nothing we can’t handle.

For me, every day is an opportunity to learn, grow, and take on new challenges. I believe that failure is an essential part of success: You must lose your balance to learn how to stand.

## My Tech Stack 🛠️
- **Languages:** TypeScript, JavaScript (ES2015+), Python, Golang, PHP (Symfony), Cypher (Neo4j-specific), Node.js as well as some 'lower level' languages like C.
- **Frontend:** React / Next.js, Vue / Nuxt, TailwindCSS, Bootstrap.
- **Backend:** Node.js, Laravel, Symfony, Python.
- **Databases:** Neo4j, PostgreSQL, Redis, MySQL 8+.
- **DevOps & Tools:** Docker, Linux (Debian, RHEL, Alpine), CI/CD (GitLab), Ansible, PHPUnit, Selenium, Symfony API Platform.
